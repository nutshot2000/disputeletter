## 🛠️ Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Google Gemini AI
- Stripe Payments
- jsPDF for PDF generation

## 🏃 Run Locally

**Prerequisites:** Node.js 18+

1. Clone and install:
   ```bash
   git clone https://github.com/nutshot2000/disputeletter.org.git
   cd disputeletter.org
   npm install
   ```

2. Set environment variables in `.env.local`:
   ```
   GEMINI_API_KEY=your_key_here
   STRIPE_SECRET_KEY=sk_test_...
   VITE_STRIPE_PUBLISHABLE_KEY=pk_test_...
   ```

3. Run the app:
   ```bash
   npm run dev
   ```
